![39C3 Artwork Generator Demo](.github/screen.gif)

## Development

### Prerequisites

- Node.js

### Setup

```bash
npm install
```

### Development Server

```bash
npm run dev
```

Starts Vite development server with hot module replacement at `http://localhost:5173`

### Production Build

```bash
npm run build
```
